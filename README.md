# Bellabeat Case Study (Google Data Analytics Capstone Project)

## 📌 Project Overview

This case study is part of the Google Data Analytics Professional Certificate. The goal is to analyze smart device usage data to gain insights into how Bellabeat, a high-tech manufacturer of health-focused products for women, can grow its business.

---

## 📊 Business Task

Analyze user data from smart devices to identify trends in physical activity, sleep, and health behavior. Provide recommendations to help Bellabeat improve their marketing strategy.

---

## 🧩 Data Source

The dataset used is public Fitbit data collected from 30 women over 31 days.  
**Files used:**

- `dailyActivity_merged.csv`
- `sleepDay_merged.csv`

Source: [FitBit Fitness Tracker Data](https://www.kaggle.com/datasets/arashnic/fitbit)
